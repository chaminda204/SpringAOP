This contains the set of instructions required to build and run spring AOP sample.

Building the Project.
====================

Pre-requisites
--------------
JDK 1.6.x or higher
Maven 2 or higher
java_home and m2_home has been set.

Steps:
------
1 - download the project.
2 - Go to the project home directory of SpringAOP, and run 
    $ mvn clean package

Running the application.
========================
Go to the test package and run AOPTest.java
